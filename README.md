## Países

Base de dados em SQL com nomes de 249 países (incluindo territórios, departamentos, regiões, coletividades etc.) em diversas línguas e também na língua nativa do respectivo país. Inclui os códigos dos países em ISO 3166-1 numérico (chave primária), ISO 3166-1 alfa-2 e ISO 3166-1 alfa-3.

### Línguas (código ISO 639-1):
